mongoid_tree

A tree structure for Mongoid documents using the materialized path pattern

Requirements

  • mongoid (>= 2.0.0.beta9)

Install

In order to get the latest development version of mongoid_tree: 

gem "mongoid_tree", :git => "git://github.com/benedikt/mongoid_tree"

Finally, run 

bundle install

Usage

Read the API documentation at benedikt.github.com/mongoid_tree and take a look at the Mongoid::Tree module 

require 'mongoid/tree'

class Node
  include Mongoid::Document
  include Mongoid::Tree
end
